hashable_containers

class hashable_containers.HGraph(incoming_graph_data=None, **attr)

networkx.Graph subclass which uses hmap containers instead of python dicts. Because hmap objects are hashable, the graph object itself can be hashed. A well-behaved __eq__ function is also defined, using the same data (graph, _node, _adj) as hash function.
